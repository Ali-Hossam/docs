# Contributing to Pipeless ü§ù

Thanks for your interest in contributing to Pipeless! Contributions are welcome and encouraged. While we're working on creating detailed contributing guidelines, here are a few general steps to get started:

import { Callout } from 'nextra/components'

<Callout>
Please note that for major changes or new features, it's a good idea to discuss them in a [GitHub discussion](https://github.com/miguelaeh/pipeless/discussions) first so we can coordinate efforts.
</Callout>

1. Fork the [repository](https://github.com/miguelaeh/pipeless).
1. Create a new branch:

    ```bash copy
    git checkout -b feature-branch`.
    ```

1. Make your changes and commit them:

    ``` bash copy
    git commit -m 'Add new feature'
    ```

1. Push your changes to your fork:

 ```bash copy
 git push origin feature-branch
 ```

1. Open a pull request describing your changes.

We appreciate your help in making this project better!
